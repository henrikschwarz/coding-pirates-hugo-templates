    Installer Arduino IDE. #

    For at programmere arduino skal vi bruge Arduino IDE. Du kan hente Arduino IDE på det på det her link. Du skal ikke hente Windows Store versionen, fordi den laver segmentationsfejl når man skal upload på nogen boards.
